Intelli-Light

Reinforcement Learning Traffic Signal Control

A reinforcement learning approach to adaptive traffic signal control trained in SUMO and evaluated against Fixed Timer and Max-Pressure.

The Problem

Traditional traffic signals rely on fixed timers that cannot adapt to real-time congestion. This leads to inefficient signal timing and unnecessary waiting during peak hours. This project explores how reinforcement learning can dynamically allocate green signal durations based on live traffic conditions to improve traffic flow and reduce congestion.

System Architecture

SUMO Traffic Simulation
↓
Traffic State Extraction (queues, waiting time, phase state)
↓
PPO Reinforcement Learning Agent
↓
Adaptive Signal Controller
↓
Adaptive Traffic Signal Optimization

Performance Results

Evaluation was conducted across multiple simulation episodes using three traffic scenarios: Morning Rush, Evening Rush, and Weekend traffic patterns. IntelliLight is compared against two widely used baseline controllers: Fixed Timer and Max-Pressure.

Key Result: IntelliLight reduces average vehicle waiting time by ~40% while maintaining comparable throughput to traditional traffic controllers.
